# 輸出格式規範

## 純文字日報格式（LINE 分享用）

禁止使用任何 Markdown 符號（#, ##, **, __, > 等）

```
📅【金蹦社群幣圈日報｜YYYY-MM-DD】

📈 昨日市場總結：
（1-2句精煉總結）

💡 三大關鍵資訊：

01｜🔥【分類標籤】核心事件標題
2-3句說明事件內容，並點出對加密貨幣市場的潛在影響。

02｜🔥【分類標籤】核心事件標題
2-3句說明事件內容，並點出對加密貨幣市場的潛在影響。

03｜🔥【分類標籤】核心事件標題
2-3句說明事件內容，並點出對加密貨幣市場的潛在影響。
```

## 分類標籤對照

- 地緣政治 ⚔️
- 幣圈安全 🛡️
- 機構採用 📈
- 監管政策 ⚖️
- 總經數據 📊
- DeFi 動態 🔗
- 產品發布 🚀
- 市場行情 💹

## 流光字卡內容（精簡版）

只取以下欄位：
- 標題 + 日期
- 核心摘要標籤
- 3 條新聞的序號、Emoji、分類標籤、核心事件標題（一行）
- 無浮水印
